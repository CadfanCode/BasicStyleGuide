Panels

Panels are used to group related content. Choose the correct type based on the nature of the content.

Successful action

The operation was completed successfully. All changes have been saved and the system is updated.

HTML Source:


<div class="panel panel-positive">
  <div class="panel-header">Successful action</div>
  <div class="panel-body">
    <p>The operation was completed successfully...</p>
  </div>
</div>

Associated CSS:


.panel-positive .panel-header {
  background-color: #40B45C;
  color: #FFFFFF;
}
.panel-positive .panel-body {
  background-color: #F3FDF5;
  color: #1A472A;
  border: 1px solid #40B45C;
  border-top: none;
}
