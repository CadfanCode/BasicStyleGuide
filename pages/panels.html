Paneler

Paneler används för att gruppera relaterat innehåll. Välj rätt typ baserat på innehållets karaktär.

Lyckad åtgärd

Operationen har genomförts framgångsrikt. Alla ändringar har sparats och systemet är uppdaterat.

HTML Source:


<div class="panel panel-positive">
  <div class="panel-header">Lyckad åtgärd</div>
  <div class="panel-body">
    <p>Operationen har genomförts framgångsrikt...</p>
  </div>
</div>

Associated CSS:


.panel-positive .panel-header {
  background-color: #40B45C; 
  color: #FFFFFF; 
}
.panel-positive .panel-body {
  background-color: #F3FDF5; 
  color: #1A472A; 
  border: 1px solid #40B45C;
  border-top: none;
}
