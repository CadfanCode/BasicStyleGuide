Buttons

Below are examples of buttons that can be used in your interface, categorized by their intended purpose.

Standard Button

HTML Source:

<button type="button" class="button-standard">Click Me!</button>

Associated CSS:


.button-standard {
    background-color: #003366;
    color: #ffffff;
}

.button-standard:hover,
.button-standard:focus {
    background-color: #004c8c;
    box-shadow: 0 6px 8px 0 rgba(0,0,0,0.5);
    transform: translateY(-1px);
}

Function Button

HTML Source:

<button type="button" class="button-function">Print</button>

Associated CSS:


        .button-function {
    background-color: #ffffff;
    color: #003366;
    border: 1px solid #003366;
    box-shadow: none;
}

.button-function:hover,
.button-function:focus {
    background-color: #f0f8ff;
    color: #0066cc;
    border-color: #0066cc;
}

Best practice:

  • Use a class when you want multiple buttons to share the same style.
  • Use a unique ID when you need to target a specific button with JavaScript or unique styling.
  • Always include type="button", type="submit", or type="reset" to ensure expected behavior.
