Color Usage Examples

This page demonstrates how the colors from the color palette should be used in different elements of your web design. The colors are used for text, backgrounds, buttons, links, and form inputs.

1. Background Colors

Primary Background Color: Dark Blue (#003366)

This color should be used for primary sections like headers or footers to provide strong contrast with text.

Secondary Background Color: Medium Blue (#0066cc)

This color is ideal for buttons or secondary call-to-action areas.

Tertiary Background Color: Light Blue (#66aaff)

Use this color for accent sections or hover states on buttons.

Accent Background Color: Sky Blue (#87ceeb)

This color works well for highlighting important elements like call-to-action buttons.

Background Color: Light Grayish Blue (#f0f8ff)

This is the main background color for your site, providing a soft, neutral backdrop.

2. Text Colors

Primary Text Color: Dark Blue (#003366)

This color should be used for headings and body text, as it provides excellent contrast against light backgrounds.

Secondary Text Color: Medium Blue (#0066cc)

Used for links or emphasis text. This color ensures readability on lighter backgrounds.

Text Color: White (#ffffff)

Use this text color on dark background elements such as navigation bars, footers, and dark buttons.
