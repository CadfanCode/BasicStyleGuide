Färgschema — snabböversikt

Här visas godkända användningsmönster för paletten. All text som visas i fälten uppfyller WCAG AAA eller har en förklaring varför en färg endast får användas dekorativt/akcentmässigt.

#003366

Använd som primär bakgrund — text: vit (AAA).

Namn: Primary / Dark Blue

Rekommendation: Huvudrubriker, nav, fot. Kontrast 12.6:1 mot vit text (AAA).

#0066cc

Endast dekor / knapp-fyllnad

Namn: Secondary / Medium Blue

Rekommendation: Använd som accent eller knappbakgrund. Inte som bakgrund för paragraftext om AAA krävs. För text på denna bakgrund använd istället ett högkontrast-etikett (t.ex. vit text + extra outline) eller placera text utanför.

#66aaff

Använd: accent/hover. Inre text: #003366 (AA). För AAA placera text utanför.

Namn: Tertiary / Light Blue

Rekommendation: Accentytor, hover-states. Inre text når endast AA-kontrast (5.3:1) — flytta viktig text utanför för AAA.

#87ceeb

Använd som accent — text: #003366 (AAA).

Namn: Accent / Sky Blue

Rekommendation: CTA-bakgrunder med mörk text inuti (kontrast 7.2:1). Bra för viktiga knappar när vit text inte passar.

#f0f8ff

Huvudbakgrund — text: #003366 (AAA).

Namn: Light / Very Pale Blue

Rekommendation: Sidbakgrunder och kort-ytor — använd mörk text (#003366) för AAA-läsbarhet.

Snabb-regler

  • #003366 = primär; trygg för rubriker och vit text (AAA).
  • #0066cc = dekor/knappfyllnad — undvik som textbakgrund för AAA.
  • #87ceeb = bra CTA med mörk text (AAA).
  • Använd tydliga fokus-stilar och aria-describedby på färgprover för skärmläsare.
