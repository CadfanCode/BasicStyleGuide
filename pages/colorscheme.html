Color scheme — quick overview

Approved usage patterns for the palette are shown here. All text shown in the fields meets WCAG AAA or has an explanation why a color may only be used decoratively/as an accent.

#003366

Use as primary background — text: white (AAA).

Name: Primary / Dark Blue

Recommendation: Main headings, nav, footer. Contrast 12.6:1 against white text (AAA).

#0066cc

Only decoration / button fill

Name: Secondary / Medium Blue

Recommendation: Use as accent or button background. Not as background for paragraph text if AAA is required. For text on this background use a high-contrast label instead (e.g. white text + extra outline) or place text outside.

#66aaff

Use: accent/hover. Inner text: #003366 (AA). For AAA place text outside.

Name: Tertiary / Light Blue

Recommendation: Accent areas, hover-states. Inner text reaches only AA contrast (5.3:1) — move important text outside for AAA.

#87ceeb

Use as accent — text: #003366 (AAA).

Name: Accent / Sky Blue

Recommendation: CTA backgrounds with dark text inside (contrast 7.2:1). Good for important buttons when white text does not fit.

#f0f8ff

Main background — text: #003366 (AAA).

Name: Light / Very Pale Blue

Recommendation: Page backgrounds and card surfaces — use dark text (#003366) for AAA readability.

Quick rules

  • #003366 = primary; safe for headings and white text (AAA).
  • #0066cc = decoration/button fill — avoid as text background for AAA.
  • #87ceeb = good CTA with dark text (AAA).
  • Use clear focus styles and aria-describedby on color samples for screen readers.
