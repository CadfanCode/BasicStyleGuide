Color scheme — quick overview

Approved usage patterns for the palette are shown here. All text shown in the fields meets WCAG AAA or has an explanation why a color may only be used decoratively/as an accent.

#003366

Use as primary background.

Name: Primary / Dark Blue

Recommendation: Main headings, nav, footer.

#0066cc

Only decoration / button fill

Name: Secondary / Medium Blue

Recommendation: Use as accent or button background.

#66aaff

Use for hover.

Name: Tertiary / Light Blue

Recommendation: Hover-states.

#87ceeb

Alternative button colour. (AAA).

Name: Accent / Sky Blue

Recommendation: Alternative button colour.

#f0f8ff

Main background

Name: Light / Very Pale Blue

Recommendation: Page backgrounds.

Quick rules

  • #003366 = primary; safe for headings and white text (AAA).
  • #0066cc = decoration/button fill — avoid as text background for AAA.
  • #87ceeb = good CTA with dark text (AAA).
