Navigation (nav)

The <nav> element represents a section of the page that contains navigation links. Below are examples of a simple navigation bar and a vertical sidebar navigation.

Simple Navigation Bar

This navigation style is suitable for top-level links on a page.

  • Home
  • Products
  • About
  • Contact

HTML source:


<nav class="nav-bar">
    <ul>
        <li><a href="#">Home</a></li>
        <li><a href="#">Products</a></li>
        <li><a href="#">About</a></li>
        <li><a href="#">Contact</a></li>
    </ul>
</nav>

Associated CSS:


        .nav-bar {
    background-color: #003366;
    text-align: center;
    padding: 10px 0;
    border-radius: 6px;
    margin-bottom: 30px;  
}

.nav-bar ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
    display: inline-flex;
}

.nav-bar li {
    margin-right: 15px;
}

.nav-bar a {
    display: block;
    padding: 12px 20px;
    color: white;
    text-decoration: none;
    font-size: 16px;
    font-weight: 500;
    border-radius: 4px;
    transition: background-color 0.3s ease, padding-left 0.3s ease;
}

.nav-bar a:hover {
    background-color: #0066cc;
    padding-left: 15px; 
}

Sidebar Navigation

This style works well for dashboards or pages with multiple subsections.

  • Dashboard
  • Reports
  • Settings
  • Logout

HTML source:


<nav class="nav-sidebar">
    <ul>
        <li><a href="#">Dashboard</a></li>
        <li><a href="#">Reports</a></li>
        <li><a href="#">Settings</a></li>
        <li><a href="#">Logout</a></li>
    </ul>
</nav>

Associated CSS:

 
    .sidebar-nav-example {
    margin-top: 40px; /
}

.nav-sidebar {
    width: 33.3%;
    background-color: #003366;
    padding-top: 20px;
    box-sizing: border-box;
    border-radius: 6px;
}

.nav-sidebar ul {
    list-style-type: none;
    margin: 0;
    padding: 0;
}

.nav-sidebar li {
    margin: 10px 0;
}

.nav-sidebar a {
    display: block;
    padding: 12px 20px;
    color: white;
    text-decoration: none;
    font-size: 16px;
    font-weight: 500;
    transition: background-color 0.3s ease, padding-left 0.3s ease;
}

.nav-sidebar a:hover {
    background-color: #0066cc;
    border-left: 5px solid #ffffff;
    padding-left: 25px; 
}

Good to know:

<nav> defines a navigation section that contains major navigation links.

<ul> defines an unordered list of links.

<li> defines each list item in the menu.

<a> defines the clickable link itself.
