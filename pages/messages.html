Messages & Alerts

This page demonstrates different types of messages that can appear in your interface, including information, success, warning, and error messages. You can also test them using the demo buttons below.

Interactive Demo

Information Message

ℹ️ This is an informational message to guide the user.

HTML source:

<div class="message info">
    <p>ℹ️ This is an informational message to guide the user.</p>
</div>

Associated CSS:

.message.info {
    background-color: #e6f0ff;
    border-left: 4px solid #003366;
    color: #003366;
    padding: 12px 15px;
    border-radius: 6px;
    margin-bottom: 20px;
}
