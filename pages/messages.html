Messages & Alerts

This page demonstrates different types of messages that can appear in your interface, including information, success, warning, and error messages.

Information Message

This is an informational message to guide the user.

Success Message

Your action was completed successfully!

Warning Message

Warning! Please check your input before proceeding.

Error Message

⚠️ Error! Something went wrong, please try again.

HTML source for Error Message:

<div class="message error">
    <p>⚠️ Error! Something went wrong, please try again.</p>
</div>

Associated CSS:

.message.error {
    background-color: #ffe6e6;
    border-left: 4px solid #cc0000;
    color: #003366;
    padding: 12px 15px;
    border-radius: 6px;
    margin-bottom: 20px;
}
