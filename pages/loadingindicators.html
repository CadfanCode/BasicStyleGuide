Loading Indicators

Below are examples of loading indicators that can be used in your interface.

Spinner Loading Indicator

HTML source:

<div class="loading-spinner"></div>

Associated CSS:


    .loading-spinner {
        width: 50px;
        height: 50px;
        border: 5px solid #f3f3f3;
        border-top: 5px solid #003366;
        border-radius: 50%;
        animation: spin 1s linear infinite;
        margin: 20px 0;
    }

    @keyframes spin {
        0% { transform: rotate(0deg); }
        100% { transform: rotate(360deg); }
    }

Progress Bar Loading Indicator

HTML source:

<div class="loading-progress-container">
    <div class="loading-progress"></div>
</div>

Associated CSS:


        .loading-progress {
            width: 0%;
            height: 20px;
            background-color: #0066cc; 
            animation: progress 3s ease-in-out forwards;
        }

        @keyframes progress {
            0% { width: 0%; }
            100% { width: 100%; }
        }

Best practice:

  • Use a spinner for short, indeterminate loading states.
  • Use a progress bar for longer loading operations where progress can be estimated.
  • Always ensure to hide the loading indicator when the action is complete or has failed.
